# Archive

Minimalist writing app with AI formatting and real-time cloud sync.

→ [archive.tirup.in](https://archive.tirup.in)

## Stack

- **Next.js 15** — App Router
- **TipTap** — Rich text editor
- **Firebase** — Auth, Firestore
- **Gemini** — AI formatting via Firebase AI SDK
- **Vercel** — Hosting

## Development

```bash
npm install
npm run dev
```
